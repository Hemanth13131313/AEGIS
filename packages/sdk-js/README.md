# aegis-sdk-js

Optional SDK for applications that want SDK-based integration (as opposed to the default transparent-proxy model).

> ⚠️ **Status:** Planned for Phase 8. This SDK is not required for basic AEGIS protection — the default integration is transparent proxy (no SDK changes needed in your app).

## Planned Features
- Node.js client for the AEGIS management API
- Helper decorators/functions for explicit policy annotation
- Test utilities for integration testing against a local AEGIS instance
